# 205. Isomorphic Strings 同构字符串

**<font color=red>难度: Easy</font>**

## 刷题内容

> 原题链接

* https://leetcode-cn.com/problems/isomorphic-strings/

> 内容描述

```
给定两个字符串 s 和 t，判断它们是否是同构的。

如果 s 中的字符可以被替换得到 t ，那么这两个字符串是同构的。

所有出现的字符都必须用另一个字符替换，同时保留字符的顺序。两个字符不能映射到同一个字符上，但字符可以映射自己本身。

示例 1:

输入: s = "egg", t = "add"
输出: true

示例 2:

输入: s = "foo", t = "bar"
输出: false

示例 3:

输入: s = "paper", t = "title"
输出: true

说明:
你可以假设 s 和 t 具有相同的长度。
```

## 解题方案

> 方法一：使用set集合，如果是同构的话，方法一里的三个集合长度会是一致的。这个还复杂化了。
>

```python
class Solution:
    def isIsomorphic(self, s: str, t: str):
        # s, t = "paper", "title"
        tuple_set = set()  # 复杂了，可以直接 set(zip(s, t))

        for s_char, t_char in zip(s, t):
            tuple_set.add((s_char, t_char))
        # [('p', 't'), ('a', 'i'), ('p', 't'), ('e', 'l'), ('r', 'e')]
        # {('a', 'i'), ('r', 'e'), ('e', 'l'), ('p', 't')}

        # 然后判断 三个集合长度是否相等
        len1 = len(set([i[0] for i in tuple_set]))
        len2 = len(set([i[1] for i in tuple_set]))
        return len1 == len2 == len(tuple_set)
```



> 方法二：一行版本。上面构造集合的方法还复杂了。同构的话，set()后的长度都一致。

```python
class Solution:
    def isIsomorphic(self, s: str, t: str):
        return len(set(zip(s,t))) == len(set(s)) == len(set(t))
```



> 方法三：用字典记录下对应的映射，但是只能单方面限制，所以我们要同时确保s 和 t 是相互isomorphic的。
>

```python
class Solution:
    def isIsomorphic(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """

        def isoHelper(s1, s2):
            lookup = {}
            for i, c in enumerate(s1):
                if s2[i] in lookup:
                    if c != lookup[s2[i]]:  # 一个字符映射了两个
                        return False
                else:
                    lookup[s2[i]] = c
            print(lookup)
            return True

        return isoHelper(s, t) and isoHelper(t, s)
```



> 方法四：跟方法三同理，两个字典相互判断映射关系。

class Solution:
    def isIsomorphic(self, s: str, t: str) -> bool:
        s_dict, t_dict = {}, {}

        for i in range(len(s)):
            if s[i] not in s_dict:
                s_dict[s[i]] = t[i]
            if t[i] not in t_dict:
                t_dict[t[i]] = s[i]

            if s_dict[s[i]] != t[i] or t_dict[t[i]] != s[i]:
                return False

        return True

```python
class Solution:
    def isIsomorphic(self, s: str, t: str) -> bool:
        s_dict, t_set = {}, set()  # s->t,t
        for i in range(len(s)):
            if s[i] in s_dict:
                # 映射不到, 一个将映射多个了
                if s_dict[s[i]] != t[i]:
                    return False
            elif t[i] in t_set:
                # s, t = "bar", "foo", 一个将映射多个了
                return False
            else:
                s_dict[s[i]] = t[i]
                t_set.add(t[i])
        return True
```

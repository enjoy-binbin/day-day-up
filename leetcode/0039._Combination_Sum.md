# 39. 组合总合 Combination Sum

**<font color=red>难度: Medium</font>**

## 刷题内容

> 原题链接

* https://leetcode-cn.com/problems/combination-sum/

> 内容描述

```
给定一个无重复元素的数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

candidates 中的数字可以无限制重复被选取。

说明：
所有数字（包括 target）都是正整数。
解集不能包含重复的组合。 

示例 1:
输入: candidates = [2,3,6,7], target = 7,
所求解集为:
[
  [7],
  [2,2,3]
]

示例 2:
输入: candidates = [2,3,5], target = 8,
所求解集为:
[
  [2,2,2,2],
  [2,3,3],
  [3,5]
]
```

## 解题方案

> 方法一：dfs回溯决策树。区别于46题全排列，还是有点变通
>

```python
class Solution(object):
    def combinationSum(self, candidates, target):
        """
        :type candidates: List[int]
        :type target: int
        :rtype: List[List[int]]
        """
        candidates.sort()
        result = []
        self.helper(candidates, target, [], result)
        return result

    def helper(self, candidates, target, paths, result):
        if sum(paths) > target:
            # 递归终止条件
            return

        if sum(paths) == target:
            # 回溯到了正确结果
            result.append(paths[:])
            return

        for num in candidates:
            # 可以适当剪枝
            if target - num < 0:
                continue

            # 防止重复, 不让其找在当前元素之前的元素
            if paths and paths[-1] > num:
                continue

            # 回溯的模板, 选择路径/递归/撤销路径
            paths.append(num)
            self.helper(candidates, target, paths, result)
            paths.remove(num)
```
